OpenBLAS_bench
==============

Implementation and benchmark of various matrix-matrix multiplication algorithms using function calls to OpenBLAS.

Dependencies
-------------
- GCC
- OpenBLAS

Building
-------------

<pre>
$ make
</pre>

Usage
------

To run benchmark utlity:
<pre>
$ ./matrix_bench
</pre>

To save results in CSV file:
<pre>
$ ./matrix_bench > benchmark.csv
</pre>
